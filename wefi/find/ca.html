Trobeu les vostres contrasenyes de Wi-Fi desades a l'iPhone o l'iPad

A iOS 16+, iPadOS 16.1+, podeu trobar contrasenyes de Wi-Fi desades.

Per a la xarxa connectada actualment

  1. A l'aplicació Configuració, toqueu Wi-Fi.
  2. Cerqueu la xarxa Wi-Fi que utilitzeu actualment.
  3. Toqueu el botó d'informació del cercle després del nom de WiFi
  4. Toqueu el camp Contrasenya.
  5. Utilitzeu la vostra contrasenya, Face ID o Touch ID per desbloquejar i revelar la contrasenya WiFi.

 

Per a xarxes prèviament connectades

A l'aplicació Configuració, toqueu Wi-Fi i, a continuació, toqueu Edita a l'extrem superior dret per veure les xarxes a les quals es va unir el vostre dispositiu anteriorment.

Podeu repetir el pas 3 al pas 5 anterior per obtenir la contrasenya de Wi-Fi desada.
